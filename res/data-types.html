Data Types in C

Learn about C’s fundamental, derived, and user-defined data types, their sizes, and examples of usage.

Overview

Every variable in C has a data type, which determines the size and type of data that can be stored in it.

  • Basic (Primitive) Data Types: int, char, float, double
  • Derived Data Types: arrays, pointers, structures, unions
  • User-defined Data Types: typedef, enum, struct, union

Example: Printing sizes and ranges

#include <stdio.h>
#include <limits.h>
#include <float.h>

int main(void) {
    printf("Size of int: %zu bytes\n", sizeof(int));
    printf("INT_MIN = %d, INT_MAX = %d\n", INT_MIN, INT_MAX);
    printf("Size of float: %zu bytes, FLT_MAX = %e\n", sizeof(float), FLT_MAX);
    printf("Size of double: %zu bytes, DBL_MAX = %e\n", sizeof(double), DBL_MAX);
    return 0;
}

Derived Data Types

  • Arrays: Collection of elements of same type.
  • Pointers: Store memory addresses.
  • Structures: Combine variables of different types.
  • Unions: Share memory among members.
// Example of an array and a pointer
int marks[5] = {90, 80, 70, 60, 50};
int *ptr = marks; // pointer to first element
printf("First mark = %d\n", *ptr);

User-defined Data Types

// enum example
enum Week { MON, TUE, WED, THU, FRI, SAT, SUN };

// typedef example
typedef unsigned long ulong;

// struct example
struct Student {
    int id;
    char name[20];
};

Type modifiers

Modifiers alter the data type’s size and range:

  • signed, unsigned
  • short, long
unsigned int u = 40000;
short int s = -10;
long double x = 3.14;

Best Practices

  • Use int for general integer operations.
  • Use float or double for real numbers (prefer double for precision).
  • Use unsigned types only when negative values are impossible.
  • Always use sizeof() to write portable programs.
