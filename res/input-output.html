Input and Output in C

Learn about standard input and output functions such as printf, scanf, getchar, putchar, gets, and puts with examples and best practices.

Introduction

The C Standard Library provides functions for performing input and output operations through stdio.h. The most common functions are printf() and scanf() for formatted I/O.

printf() — Output function

printf() is used to display formatted output to the standard output (usually the screen).

#include <stdio.h>
int main(void) {
    int age = 24;
    float height = 1.74;
    printf("Age: %d years\n", age);
    printf("Height: %.2f meters\n", height);
    return 0;
}
  • %d – integer
  • %f – float
  • %c – character
  • %s – string
  • %.2f – float with 2 decimal precision

scanf() — Input function

scanf() reads formatted input from the user through standard input (keyboard).

#include <stdio.h>
int main(void) {
    int age;
    float height;
    printf("Enter your age and height: ");
    scanf("%d %f", &age, &height);
    printf("You entered: Age=%d, Height=%.2f\n", age, height);
    return 0;
}
Always use the address-of operator (&) before variable names in scanf() (except for strings).

Character I/O: getchar() and putchar()

#include <stdio.h>
int main(void) {
    char ch;
    printf("Enter a character: ");
    ch = getchar();  // reads a single character
    printf("You entered: ");
    putchar(ch);     // prints the character
    putchar('\n');
    return 0;
}

String I/O: gets() and puts()

#include <stdio.h>
int main(void) {
    char name[50];
    printf("Enter your name: ");
    fgets(name, sizeof(name), stdin); // safer than gets()
    printf("Hello, ");
    puts(name);
    return 0;
}
Avoid using gets() as it is unsafe and removed from modern C standards. Use fgets() instead.

Example: Basic Calculator using scanf and printf

#include <stdio.h>
int main(void) {
    double a, b;
    char op;
    printf("Enter expression (e.g. 5 + 3): ");
    scanf("%lf %c %lf", &a, &op, &b);

    switch (op) {
        case '+': printf("%.2f\n", a + b); break;
        case '-': printf("%.2f\n", a - b); break;
        case '*': printf("%.2f\n", a * b); break;
        case '/':
            if (b != 0) printf("%.2f\n", a / b);
            else printf("Error: Division by zero\n");
            break;
        default: printf("Invalid operator!\n");
    }
    return 0;
}

Best Practices

  • Always match format specifiers with correct data types.
  • Use fgets() for string input to avoid buffer overflow.
  • Check return values of scanf() and printf() for robust error handling.
  • Print clear prompts before input.
