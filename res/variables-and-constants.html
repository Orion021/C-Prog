Variables and Constants

Learn what variables and constants are in C, how to declare and use them, and understand their rules, scope, and lifetime.

What are Variables?

A variable is a named location in memory that stores data. Its value can change during program execution.

int age = 24;      // integer variable
float height = 1.74; // floating point variable
char grade = 'A';   // character variable
Variables must be declared before use, specifying their data type and optionally an initial value.

Rules for naming variables

  • Can contain letters, digits, and underscore (_).
  • Must begin with a letter or underscore (not a digit).
  • C is case-sensitive (Score and score are different).
  • No spaces or special symbols allowed.
  • Should not be a C keyword (e.g., int, return).

Variable declaration and initialization

Variables can be declared and initialized in several ways:

int a;          // declaration only
int b = 10;     // declaration + initialization
int x = 5, y = 7, z = 9; // multiple declarations

Uninitialized variables may contain garbage values and cause undefined behavior if used before assignment.

Types of variables based on scope

  • Local variables: Declared inside a function/block, accessible only within it.
  • Global variables: Declared outside functions, accessible throughout the program.
  • Static variables: Retain their value between function calls.
#include <stdio.h>

int g = 10; // global variable

void func() {
    static int count = 0; // static variable
    int local = 5;        // local variable
    count++;
    printf("g=%d, count=%d, local=%d\n", g, count, local);
}

int main(void) {
    func();
    func();
    return 0;
}

Output: 

g=10, count=1, local=5
g=10, count=2, local=5
Static variables preserve their value between calls, while local variables are reinitialized every call.

Constants

A constant is a fixed value that does not change during program execution.

const float PI = 3.14159; // named constant
#define MAX 100            // symbolic constant using macro

Constants improve code readability and maintainability. You can’t assign to them after declaration.

Literals in C

  • Integer literals: e.g. 10, 0xA, 075.
  • Floating literals: e.g. 3.14, 2.5e-3.
  • Character literals: e.g. 'A', '\n'.
  • String literals: e.g. "Hello".

Example program: Area of a circle

#include <stdio.h>
#define PI 3.14159

int main(void) {
    float radius, area;
    printf("Enter radius: ");
    scanf("%f", &radius);
    area = PI * radius * radius;
    printf("Area = %.2f\n", area);
    return 0;
}

Best practices

  • Use descriptive names like totalMarks instead of tm.
  • Initialize variables as you declare them.
  • Use const or #define for values that shouldn’t change.
  • Keep variable scope as small as possible to avoid confusion.
